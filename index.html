Configuration de l'accès GitHub

Le site sauvegarde les disponibilités dans votre dépôt GitHub.

Ce jeton doit avoir les permissions d'écriture pour le dépôt latmacchiato/cal30ans.
Comment obtenir ce token ?
1. Allez sur GitHub > Settings > Developer Settings > Personal Access Tokens > Fine-grained tokens.
2. Générez un jeton avec accès en écriture sur le dépôt de votre calendrier.
3. Partagez-le avec vos amis pour qu'ils puissent s'identifier et écrire !
